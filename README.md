# Amazon Pinpoint

Amazon Pinpoint is a flexible and scalable outbound and inbound marketing communications service that enables you to engage with customers across multiple messaging channels.

## APIs

- **Amazon Pinpoint API** - The Amazon Pinpoint API enables you to create and manage marketing campaigns, send transactional messages, and define audience segments for targeted communications.
